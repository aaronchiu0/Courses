Pre Integration

Riemann Sums

The area under a curve can be split into multiple different rectangles. Adding the rectangles results in the total area. \begin{align*} y = f(x); \ a \leq x \leq b; \ \text{with $n$ sub intervals} \end{align*}

Using Left Endpoints

\begin{align*} R_l = \sum_{i = 1}^n f(x_i) \Delta x && x_i=a+i \Delta x - \Delta x && \Delta x = \frac{b - a}{n} \end{align*}

Using Right Endpoints

\begin{align*} R_r = \sum_{i = 1}^n f(x_i) \Delta x && x_i=a+i \Delta x && \Delta x = \frac{b - a}{n} \end{align*}

Using Midpoints (Average)

\begin{align*} R_m = \sum_{i = 1}^n f(\bar{x_i}) \Delta x && \bar{x_i}=a+i \Delta x -\frac{1}{2} \Delta x && \Delta x = \frac{b - a}{n} \end{align*}

When $n$ is small, this results in areas that are not taken into account resulting in an inaccurate area.

Definite Integral

When there are an infinitesimal amount of sub intervals, this results in the actual area under the curve. \begin{align*} \lim_{n\to\infty}\sum_{i = 1}^n f(x_i^*) = \int_a^b f(x)dx \end{align*} Where $\int_a^b f(x)dx$ is the definite integral of the functions running from a to b.
